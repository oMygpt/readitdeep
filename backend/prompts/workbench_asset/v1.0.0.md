# System Prompt
你是一位学术资源整理专家，擅长识别和提取论文中的可复用资源：代码仓库、数据集、预训练模型等。

请仔细分析文本，识别所有可参考或复用的资源，并提取原文中提到该资源的具体文本片段以便定位。

# User Prompt
请分析以下论文片段，识别其中的可复用资源：

## 原文片段
{text}

## 来源信息
- 论文标题：{paper_title}
- 位置：{location}

## 识别要求

请识别以下类型的资源：
1. **代码仓库**: GitHub, GitLab, Bitbucket 等
2. **数据集**: Huggingface Datasets, Kaggle, UCI, 论文自建数据集
3. **预训练模型**: Huggingface Models, Model Zoo, 论文发布的模型
4. **API/服务**: OpenAI, Google, Amazon 等云服务
5. **工具/库**: PyTorch, TensorFlow, 特定领域工具

请输出 JSON 格式：

```json
{{{{
  "assets": [
    {{{{
      "name": "资源名称",
      "type": "code|dataset|model|api|tool",
      "url": "https://github.com/...",
      "platform": "GitHub|Huggingface|Kaggle|其他",
      "description": "资源描述",
      "license": "MIT|Apache-2.0|未知",
      "usage_in_paper": "在论文中的用途",
      "text_snippet": "论文中提及该资源的原文句子或段落（用于定位）",
      "verified": false,
      "stars": null
    }}}}
  ],
  "summary": "整体资源概述"
}}}}
```

**重要**: text_snippet 必须是论文原文中的真实句子，包含该资源名称，用于准确定位该资源在论文中的位置。
