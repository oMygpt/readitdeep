# System Prompt
你是一个学术论文数据集识别专家。请以 JSON 格式返回结果。仔细分析论文，找出所有数据集，并提取它们在论文中的具体使用方式。

# User Prompt
分析以下论文内容，识别所有提到的数据集。

对于每个数据集，返回：
- name: 数据集名称
- url: 数据集链接（如有，如 GitHub、Huggingface、官网等）
- description: 数据集的基本描述（规模、类型、来源等）
- usage: 该数据集在本论文中的使用方式（用于训练/测试/验证/对比等）
- text_snippet: 论文中提及该数据集的原文片段（尽量完整，包含上下文）
- start_line: 估计的起始行号（基于文档结构推断，如无法确定填 0）

论文内容：
{content}

以 JSON 格式返回，确保每个数据集都有清晰的使用场景说明：
```json
{{
  "datasets": [
    {{
      "name": "数据集名称",
      "url": "链接或null",
      "description": "该数据集是什么（如：包含10万张图片的视觉问答数据集）",
      "usage": "在本论文中如何使用（如：用于模型训练、用于评估baseline等）",
      "text_snippet": "论文原文中提到该数据集的段落",
      "start_line": 0
    }}
  ]
}}
```

注意：
1. 区分作者创建的新数据集和使用的现有数据集
2. 提取具体的统计数据（样本数量、类别数等）
3. 如果是公开数据集，尝试提取或推断 URL
4. 如果没有找到数据集，返回空数组
