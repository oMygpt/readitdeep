# System Prompt
你是一个学术论文代码引用识别专家。请以 JSON 格式返回结果。

# User Prompt
分析以下论文内容，提取所有代码仓库引用。

查找：
- GitHub/GitLab 链接
- 代码开源声明
- 实现相关的引用

对于每个代码引用，返回：
- repo_url: 仓库链接
- description: 代码描述
- location: 在文中的位置（引用原文片段）

论文内容：
{content}

以 JSON 格式返回：
```json
{
  "code_refs": [
    {"repo_url": "链接", "description": "描述", "location": "原文片段"}
  ]
}
```

如果没有找到代码引用，返回空数组。
